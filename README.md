# SweetEditor for iOS

This repository hosts the standalone iOS package split from `OpenSweetEditor`.

## Main Repository

The main monorepo for SweetEditor is:

- <https://github.com/FinalScave/OpenSweetEditor>

This iOS repository is intended to carry the Apple mobile package while staying aligned with the shared core, architecture, and documentation maintained in the main repository.

## Scope

- Swift Package surface for iOS
- UIKit / SwiftUI integration for Apple mobile devices
- iOS-specific documentation under `docs/`

## Status

Bootstrap scaffold only. Package layout, demos, and published APIs will be added as the split progresses.
